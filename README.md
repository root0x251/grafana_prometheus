# Система мониторинга
Система предназначена для сбора, хранения, визуализации метрик с удаленных серверов, с <b>Docker<b> и сетевого оборудования UPS по протоколу <b>SNMP<b>.

### Структура проекта отслеживающей машины

> docker-compose.yml          # Основной файл оркестрации стека мониторинга
> .env_example                # Шаблон переменных окружения (не работает, не разобрался я)

> prometheus/
>> prometheus.yml.tmpl     # Шаблон конфига с переменными ${SERVER_IP_...}. Убрать .tmpl и изменить на свои параметры

> snmp/                                       
>> snmp.yml.tmpl           # Шаблон конфига SNMP. Убрать .tmpl и изменить на свои параметры.

> to_monitored_server/        # Директория с агентами для отслеживаемого сервера
>> docker-compose.yml
